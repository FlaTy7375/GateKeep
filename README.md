# GateKeep - Система управления доступом пользователей

## Общее описание
**GateKeep** — это полноценное веб-приложение для регистрации, аутентификации и администрирования пользователей. Проект разделен на две независимые части: **Frontend** (React + Vite) и **Backend** (Node.js + Express + PostgreSQL).

- Безопасная аутентификация через JWT-токены.
- Хэширование паролей (bcrypt).
- Асинхронное подтверждение email.
- Панель управления пользователями с возможностью множественного выбора, блокировки, разблокировки и удаления.
- Автоматическая проверка состояния аккаунта (блокировка/удаление) при каждом API запросе.

---

## Требования для запуска
- **Node.js** (версия 18 и выше)
- **PostgreSQL** база данных (можно использовать локальную БД или облачную, например, Neon, Supabase)

## 🛠 Запуск Backend (Серверная часть)

1. Откройте терминал и перейдите в директорию бэкенда:
   ```bash
   cd backend
   ```

2. Установите все необходимые зависимости:
   ```bash
   npm install
   ```

3. Создайте конфигурационный файл `.env` в папке `backend` и заполните его по примеру:
   ```env
   PORT=5000
   DATABASE_URL=postgresql://пользователь:пароль@хост/база_данных?sslmode=require
   JWT_SECRET=ваш_секретный_ключ_для_токенов
   
   # Настройки для отправки писем подтверждения (опционально для тестов)
   EMAIL_USER=ваш_email@gmail.com
   EMAIL_PASS=ваш_пароль_приложения

   # Обязательно для продакшена (чтобы ссылки в письмах вели на реальный домен)
   BASE_URL=https://api.yourdomain.com
   FRONTEND_URL=https://yourdomain.com
   ```
   *(Примечание: `EMAIL_PASS` — это не пароль от почты, а специальный App Password, если вы используете Gmail).*

4. Запустите сервер:
   - Для разработки (с автоперезагрузкой при изменениях кода):
     ```bash
     npm run dev
     ```
   - Для продакшена:
     ```bash
     npm start
     ```
   
   *Сервер по умолчанию будет доступен по адресу: http://localhost:5000*

## 🎨 Запуск Frontend (Клиентская часть)

1. Откройте **новый** терминал и перейдите в директорию фронтенда:
   ```bash
   cd frontend
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл `.env` в папке `frontend`. В нём указывается адрес вашего бэкенд-сервера:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

4. Запустите локальный сервер разработки:
   ```bash
   npm run dev
   ```

5. Откройте браузер и перейдите по адресу: `http://localhost:5173`

## Структура проекта
- `/backend/` — API-сервер (Node.js/Express)
  - `/routes/` — логика API (регистрация, логин, управление пользователями)
  - `/middleware/` — защита эндпоинтов и проверка статуса (JWT)
  - `db.js` — подключение к базе данных
- `/frontend/` — клиентское приложение
  - Построено по методологии **FSD (Feature-Sliced Design)**.
  - Используется React-Bootstrap для UI компонентов.
  - Включает адаптивную вёрстку для всех типов устройств.
